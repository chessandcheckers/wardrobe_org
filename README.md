## Overview

This project is created with the purpose of simplifying wardrobe clean out. 

We often buy clothes hoping we will wear them one day or save it for 
a special occasion. But we end up accumulating all these clothes, 
some we wear, some we save for our distant future, some are just 
collecting dust at the back of the closet.

So, keeping what we will actually wear and discarding or donating 
the rest is not just an storage decision it is also an economic and 
social one.
